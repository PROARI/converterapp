Bienvenido a Converter App

Converter App te permite empaquetar cualquier sitio web en una PWA (Progressive Web App) para que se comporte como una aplicación de escritorio o móvil.

1
Configura tu aplicación ingresando el enlace (URL) y seleccionando nombre, colores y logotipo.
2
Haz clic en Auto Instalar para añadir la aplicación a tu menú de inicio o escritorio directamente desde el navegador.
3
O selecciona Descargar ZIP para obtener el código fuente completo, listo para subirlo a tu propio hosting en minutos.

Nota para iOS: En dispositivos Apple (Safari), para instalar directamente haz clic en el botón de Compartir y luego en "Añadir a la pantalla de inicio".
